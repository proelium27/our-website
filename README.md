# Fixable

Web design agency site targeting local businesses. Offers website design, SEO, and ongoing maintenance — with a free mockup request form as the primary conversion goal.

## Stack

- Next.js 16, React 19, TypeScript 5
- Tailwind CSS 4, shadcn/ui, Base UI
- Hosted on Vercel via GitHub auto-deploy

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Push to `main` → Vercel auto-deploys. Do not use `vercel deploy` directly.

## Project Structure

```
src/
  app/        # Next.js app router pages
  components/ # Reusable UI components
```
